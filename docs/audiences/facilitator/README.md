## Before the Workshop

1. Complete [Pre-Deployment Guide](pre-deployment-guide.md) (day before)
2. Review [Facilitator Guide](facilitator-guide.md)
3. Print [Scoring Rubric](scoring-rubric.md)
4. Keep [Curveball Script](curveball-script.md) secret!

---

## During the Workshop

- Follow timing in [Facilitator Guide](facilitator-guide.md)
- Use [Scoring Rubric](scoring-rubric.md) for evaluation
- Announce curveball at **14:45 sharp**
- Reference [Solution Reference](solution-reference.md) for coaching

---

## After the Workshop

1. Run cleanup scripts
2. Tally final scores
3. Collect feedback forms
4. Post-workshop debrief
